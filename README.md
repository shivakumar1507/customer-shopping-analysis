# Customer Shopping Behavior Analysis

## 📊 Project Overview

This project analyzes customer shopping behavior using Python, SQL, and Power BI.

## 🛠️ Tools Used

* Python (Pandas)
* MySQL
* Power BI

## 📈 Key Insights

* Repeat customers contribute majority of revenue
* Clothing category generates highest sales
* Discounts reduce average order value
* Mid-age customers spend more
